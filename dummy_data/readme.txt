Dit is de dummy data vanuit het hoofdkantoor van klas 2WDV2.
Aangezien onze API nog niet af is hebben we dit gemaakt.
Je kan dus in je code doen alsof je dit van onze API krijgt, en zo je website aanpassen gebaseerd op onze JSON structuur.

Als je dit nog niet zelf hebt gedaan kan je ook onze "reserve-place" API call gaan gebruiken. 
Met deze call kan je dus plekken gaan reserveren. 
En met de "location-movie-data" kan je dan weer checken welke plekken nog beschikbaar zijn.
